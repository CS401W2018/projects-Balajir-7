Question1 : List out the different ways to select an item.

We can select an iteam using CSS selectors like tag name, ID, class, span tag.

Question2 : What is the difference between an ID and a Class?

A class can be used multiple HTML elements but ID can only used one HTML element.

Question3 ; What is the difference between a SPAN tag and a DIV tag?

Span tag is inline element but div tag is block-level element.

Question 4 : What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

A Div tag is devision or selection in HTML, while header,main and footer is used to design the body of the website.

Question 5: What are the different ways to select a color?

Colors can be selected by the color name or with RGB, HEX, HSL, RGBA, HSLA.
