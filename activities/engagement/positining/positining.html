Normal flow is a way to position elements on the webpage.It uses block and inline elements to Positioning.

Float is used to Positioning and formatting. It is also used to wrape text around image. We need to clear the float to avoid the content getting wrapped with the floating element.

ABSULATE is used to scrool with the pages and can change the flow, it also allows to place the element on top of each other. FIXED means the elements states in the same position even when we scrool down the page, it dosen't leave any gap in the page.RELATIVE is used to keep the element in its normal position, it dosen't affect the flow.
